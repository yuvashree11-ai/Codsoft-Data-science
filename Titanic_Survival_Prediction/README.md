Titanic Survival Prediction

Project Overview

The Titanic Survival Prediction project is a machine learning application that predicts whether a passenger survived the Titanic disaster based on passenger information such as age, gender, and ticket class.

Objective

The objective of this project is to build a predictive model that can determine the likelihood of a passenger surviving the Titanic disaster using historical data.

Dataset

This project uses the Titanic dataset, which contains information about passengers aboard the Titanic, including:

- Passenger Class (Pclass)
- Gender (Sex)
- Age
- Survival Status (Survived)

Technologies Used

- Python
- Pandas
- Scikit-learn

Machine Learning Algorithm

- Logistic Regression

Project Workflow

1. Load and explore the Titanic dataset.
2. Handle missing values in the Age column.
3. Convert categorical data into numerical format.
4. Split the dataset into training and testing sets.
5. Train the Logistic Regression model.
6. Evaluate model performance using accuracy score.

Results

The model successfully predicted passenger survival with approximately 81% accuracy.

Learning Outcomes

- Data cleaning and preprocessing.
- Handling missing values.
- Feature selection and encoding.
- Binary classification using Logistic Regression.
- Model evaluation and accuracy measurement.

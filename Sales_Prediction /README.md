Sales Prediction Using Python

Project Overview

The Sales Prediction project is a machine learning application that predicts product sales based on advertising expenditures across different media channels.

Objective

The objective of this project is to develop a regression model capable of estimating sales figures using advertising investments in TV, Radio, and Newspaper marketing campaigns.

Dataset

This project uses a sales dataset containing:

- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales Revenue

Technologies Used

- Python
- Pandas
- Scikit-learn

Machine Learning Algorithm

- Linear Regression

Project Workflow

1. Load and analyze the sales dataset.
2. Select relevant advertising features.
3. Split the dataset into training and testing sets.
4. Train a Linear Regression model.
5. Generate sales predictions for test data.

Results

The model successfully predicted sales values based on advertising expenditure and demonstrated the practical application of regression analysis.

Learning Outcomes

- Data analysis using Pandas.
- Feature selection for predictive modeling.
- Regression analysis using Linear Regression.
- Understanding relationships between advertising and sales.
- Model training and prediction.
